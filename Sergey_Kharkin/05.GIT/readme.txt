Версия 4.2 (основано на версии 4.1)
Год по умолчанию 2018
По умолчанию в СБ нет звонков
По умолчанию, второе реле включается через секунду после начала звонка на урок.
Состояние "Урок" вычисляется после подачи питания и после установки времени.
Точка на индикаторе расписания означает:
	не счетится - перемена - реле выключено
	мигает - идет урок - реле выключено
	счетится - идет урок - реле включено
